# Sudoku Game 🎮💗

## About
A GUI based Sudoku Game built in Java using Java Swing
as a Semester 3 DSA project at IOBM University.
Features a beautiful pink aesthetic theme with
real Data Structures and Algorithms implementation.

## Features
- 3 difficulty levels — Easy, Medium, Hard
- Backtracking algorithm generates valid puzzles
- Live answer checking
- Auto solve button
- Custom pink themed GUI
- Rounded buttons with hover effects
- Input validation — only numbers 1 to 9 allowed

## Language
Java

## Data Structures and Algorithms Used
- Backtracking Algorithm — generates and solves board
- Recursion — core of the solving algorithm
- Queue — removes cells based on difficulty
- HashSet — tracks removed cell positions
- ArrayList with shuffle — randomizes number placement
- 2D Arrays — stores board state and solution

## GUI Technologies
- Java Swing
- JFrame, JTextField, JButton, JComboBox
- Custom RoundedBorder class
- ActionListener and MouseListener for events
- GridLayout and BorderLayout

## What I Learned
This project taught me how backtracking algorithms
work in real applications. Implementing Queue and
HashSet for cell removal showed me how choosing
the right data structure directly affects how
clean and efficient your code becomes.
